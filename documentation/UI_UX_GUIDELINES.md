# UI/UX Guidelines

## Brand

- **Primary:** Gold `#D4AF37`
- **Background:** Beige `#F5F1E8`, White `#FFFFFF`
- **Text:** Black `#1A1A1A`, Muted `#6B6259`
- **Fonts:** Playfair Display (headings), Poppins (body)

## Components

- Use existing CSS classes from `src/styles/globals.css`
- Buttons: `.btn`, `.btn-gold`, `.btn-outline`, `.btn-dark`
- Cards: `.card-widget`, `.service-card`, `.package-card`
- Badges: `.badge-gold`, `.badge-green`, `.badge-blue`, `.badge-red`

## Layout

- Fixed navbar height: 72px (`--nav-h`)
- Max content width: 1200px
- Section padding: 100px 60px (responsive: 60px 24px)

## Animations

- Entry animations: `.animate-in`, `.delay-1` through `.delay-4`
- Hover lifts on cards: `translateY(-6px)`
